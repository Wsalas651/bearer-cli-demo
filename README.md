# Bearer CLI Demo

Este repositorio muestra cómo aplicar **Bearer CLI** para ejecutar análisis de seguridad estático (SAST) en una aplicación Node.js con código vulnerable.

## 🔧 Requisitos

- Node.js
- GitHub Actions

## 🚀 Uso

```bash
git clone https://github.com/tu-usuario/bearer-cli-demo
cd bearer-cli-demo
npm install
npm start
